# ESP32RC v1.3.0 Release Notes

**Release Date**: March 2026  
**Version**: v1.3.0 - "Precision Edition"  
**Status**: Phase 1 Complete (DS18B20 Integration) | Phase 2-3 Planned

## Overview

v1.3.0 focuses on **precision and accuracy improvements** through enhanced temperature sensing. The first phase adds DS18B20 external temperature sensor support with automatic fallback to the internal ESP32 sensor, improving fan control response time and accuracy.

## What's New

### 🌡️ Phase 1: DS18B20 External Temperature Sensor (COMPLETE ✅)

**Hardware Integration**:
- DS18B20 1-Wire protocol temperature sensor
- GPIO 19 (configurable) with 4.7kΩ pull-up resistor
- 12-bit precision: 0.0625°C resolution (vs internal ±5°C)
- Temperature range: -55°C to +125°C
- Conversion time: 750ms @ 12-bit

**Firmware Enhancements**:
- Automatic primary→fallback sensor switching
- Enhanced `readSystemTemperature()` with dual-sensor logic
- Error detection using -127.00 sentinel value
- Sensor initialization with device count detection
- Maintains backward compatibility (works without external sensor)

**Library Dependencies**:
- `paulstoffregen/OneWire@^2.3.7` - 1-Wire protocol support
- `milesburton/DallasTemperature@^3.11.0` - DS18B20 driver

**Benefits**:
✅ **10x better precision** (0.0625°C vs ±5°C)  
✅ **Faster response** (750ms conversion vs on-demand internal read)  
✅ **More stable** (external sensor away from CPU heat)  
✅ **Robust fallback** (works with or without external sensor)  
✅ **Factory calibrated** (no calibration needed)

## Configuration Changes

### New `config.h` Constants (v1.3)

```c
// External Temperature Sensor (DS18B20)
#define TEMP_SENSOR_PIN 19             // GPIO 19 for OneWire
#define TEMP_SENSOR_ENABLED 1          // Enable external sensor support
#define TEMP_SENSOR_PRECISION 12       // 12-bit (0.0625°C resolution)
#define TEMP_SENSOR_REQUEST_TIMEOUT 1000  // 1000ms timeout
#define TEMP_SENSOR_USE_EXTERNAL 1     // Use external when available
```


## Migration Guide (v1.2 → v1.3)

### No Breaking Changes ✅

v1.3.0 is **100% backward compatible** with v1.2:
- Firmware works without external DS18B20 sensor
- Falls back to internal ESP32 sensor if external unavailable
- No configuration changes required (optional enhancement)
- Existing serial commands unchanged
- All v1.2 features intact

### Optional: Add DS18B20 Sensor

To use the new external temperature sensor:

1. **Add hardware** (GPIO 19):
   ```
   ESP32 GPIO 19 (with 4.7k pull-up to 3.3V) → DS18B20 DQ pin
   3.3V → DS18B20 Vcc (red wire)
   GND → DS18B20 GND (black wire)
   ```

2. **Update firmware** to v1.3.0

3. **Check serial output**:
   ```
   > fan status
   Temperature: 42.5°C         ← Now reads from DS18B20
   Temp Range: 35°C - 50°C
   ```

If sensor unavailable, automatically reverts to internal sensor without errors.
